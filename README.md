# Отчёт о выполнении расчётно-графической работы (РГР)

## Тема проекта
Разработка многостраничного интерактивного веб-приложения (дашборда) для визуализации данных и прогнозирования стоимости алмазов методами машинного обучения.

---

## Описание проекта
Данный проект представляет собой законченное веб-приложение, написанное на языке Python с использованием фреймворка Streamlit. Приложение позволяет пользователю проводить разведочный анализ данных (EDA) и рассчитывать предполагаемую стоимость алмазов на основе их физических параметров (вес, размеры) и качественных характеристик (огранка, цвет, чистота).

---

## Структура репозитория

В репозитории находятся следующие обязательные файлы проекта:
* **app.py** — основной программный код, отвечающий за интерфейс сайта, вкладки и логику работы приложения.
* **requirements.txt** — конфигурационный файл со списком библиотек, необходимых для корректного запуска проекта на сервере.
* **Файлы моделей (*.pkl)** — сохраненные (сериализованные) обученные модели машинного обучения (Decision Tree, Random Forest, Gradient Boosting, MLPRegressor, Stacking) и объект масштабирования данных StandardScaler.
* **Модель CatBoost** — лучшая обученная модель градиентного бустинга, показавшая наивысшую точность в задаче регрессии.
* **cleaned_diamonds.csv** — предобработанная таблица с данными об алмазах, которая используется для построения графиков и вывода статистики.
* **Медиа-файлы** — изображения для оформления интерфейса (фотография профиля автора работы и иконка приложения).

---

## Функциональные возможности дашборда

Веб-приложение разделено на четыре логические страницы:
1. **Профиль:** Главная страница с информацией об авторе работы, теме РГР и графическим оформлением.
2. **Описание датасета:** Раздел со статистической сводкой по признакам и табличным превью данных.
3. **Визуализация (EDA):** Интерактивные графики распределения целевой переменной, плотности признаков и матрица корреляции для поиска зависимостей.
4. **Инференс моделей:** Рабочий модуль для предсказания цен. Поддерживает ручной ввод характеристик через ползунки (одиночный режим) и пакетную обработку готовых файлов формата *.csv (массовый режим).
